SSTすごろく V2.3.1

【修正内容】
・V2.3で「つぎのひとへ」ボタンを削除した後も、
  JavaScript側がそのボタンを参照していたため発生していた起動エラーを修正
・V2.3の機能はそのまま維持

【主な機能】
・盤面の数字を丸みのあるフォントに変更
・サイコロを振れる場面では、盤面・順番欄・余白をクリックしても振れる
・お題カード右上の✕で閉じる
・お題を閉じると自動で次の人へ
・「○○さんの ばんだよ」を画面中央に約3秒表示

【使い方】
board.pngを同じフォルダへ入れ、index.htmlを開いてください。
GitHubでは既存ファイルをすべて上書きしてください。
